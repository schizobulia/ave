# AVE
### 一个音视频处理工具

- #### 使用到的技术
    - ##### [rust](https://www.rust-lang.org)
    - ##### [gstreamer](https://gstreamer.freedesktop.org)

- ### 目录结构
    + [ave-video](./ave-video)：视频处理
    + [ave-image](./ave-image)：图片处理
    + [ave-tool](./ave-tool)：常用工具
    + [iced-style](./iced-style)：自定义iced样式库
